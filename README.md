# Portafolio Personal

Este es el repositorio de mi portafolio personal desarrollado con Next.js y TypeScript.

## Instalación y Uso

Para poder visualizar correctamente el portafolio, es necesario clonar este repositorio y seguir los pasos de instalación:

```bash
git clone git@github.com:Qu3zada22/portafolio.git   (si se hace con SSH)
cd [portafolio]
npm install
npm run dev
```

**Nota importante:** En el servidor de la clase he subido una versión comprimida (ZIP) de este portafolio, ya que el proyecto completo es demasiado pesado y quiero evitar posibles caídas del servidor. Sin embargo, para tener la experiencia completa con todas las funcionalidades, recomiendo clonar el repositorio como se indica arriba.
